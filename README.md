Brewery API magento 2 module

###Installation

Add to your magento2 composer.json:

```
{
    ...
    "repositories": [
         ...
        {
            "url": "https://github.com/c4boomb/mon-test-brewery.git",
            "type": "git"
        }
    ],
    "require": {
        ...
        "c4boomb/magento2-brewery": "dev-dev",
    },
    ...
}
```

### Requirements

Check module composer.json for requirements
